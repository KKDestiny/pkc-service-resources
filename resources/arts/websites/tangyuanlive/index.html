汤圆直播是一组由直播业务管理模块和RTMP服务模块组成的系统 简易直播平台，实现了用户管理、主播管理等完整的一套业务。

Node.js + MongoDB + Redis + Bootstrap4.0

它使用Node.js(Express框架)实现了整个系统，数据库为MongoDB+Redis， 前端框架使用了当前最为流行的Bootstrap(4.0)，并由作者进行定制化修改。

推流码认证与监控

通过OBS等推流软件，将主播的视频推送到RTMP服务器， 实现了对主播的直播。而通过推流码的认证机制，阻止了 非认证主播进行非法推流；管理员也可随时禁止直播违法内容 的主播，做到了实时监控。

Redis状态维护

增加使用Redis数据库来作为缓存服务器， 加速了一些常用数据的查询速度， 实现了业务模块和RTMP服务模块的数据交互。
